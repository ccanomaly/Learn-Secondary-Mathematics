What Is A Radical?

The general form of a radical is defined like so,
$\sqrt[n]{x}$
$\sqrt{\phantom{x}}$: Radical Symbol
$n$: Index
$x$: Radicand

What does this mean mathematically?

A radical is seen as an expression that results in a number that can be multiplied by itself n times to get the radicand $(x)$.

Example: Evaluate the radical expression $\sqrt[3]{27}$. Show Solution

To evaluate this radical, we need to come up with a number that we can mulitply itself by 3 times to get the radicand. In this case, the index is 3 and the radicand is 27, so we want to find a number that we can multiply itself by 3 times to get 27. 3 multiplied by itself 3 times is $3*3*3=27$. Hence, the answer is: $\sqrt[3]{27}=3$

Root Terminology

The conventional way to to express a radical expression out loud involves referring to the radical as "the $n^{\text{th}}$ root of $x$," where $n$ is the index and $x$ is the radicand.
For example, $\sqrt[4]{16}$ is expressed as "the $4^{\text{th}}$ root of 16."

The special case when n is 2 or 3

When you see a radical of the form $\sqrt{x}$, we call this the square root of $x$. In a square root there is no index present; by default, there is an invisible 2 there to denote that the square root is a radical with an index of 2. When a radical is in the form $\sqrt[3]{x}$, we call it the cube root of $x$.
